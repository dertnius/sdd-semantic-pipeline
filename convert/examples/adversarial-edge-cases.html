4. Custom DSL mislabeled vs. heuristic recovery

Authors set brush: java for DTK's pipe-delimited DSL. It is not Java. The converter keeps the author's brush (Tier-2, faithful-to-source):

FILE|REPLACE|<replace_file>|<source_compare_file>|<target_compare_file>|<replace_column>|<match_column>

Contrast: a brush: java block whose content is unambiguously SQL — the java-distrust heuristic should re-label it sql:

SELECT Id, Name FROM OCE__Rating__c WHERE Name != null
